# Блог на Vue 3

Мини-блог с авторизацией, созданием постов, комментариями и валидацией форм.

# Запуск проекта

```bash
npm install
npm run dev
```

## Функционал

- Авторизация
- Создание и редактирование блога
- Создание, редактирование, удаление постов
- Создание и удаление комментариев
- Пагинация на главной странице
- Валидация всех форм с подсветкой ошибок
- Адаптивный дизайн
